Futebol Feminino

Aquecimento

Anaeróbicos: Entre os tipos de exercícios físicos que merecem destaque, estão os anaeróbicos. Os movimentos exigem grande esforço dos músculos e longas séries de repetições. Eles promovem a hipertrofia muscular e aumentam a força física, o que proporciona benefícios incontestáveis para os sistemas respiratório e cardiovascular. Como exemplo, podem ser citados alguns tipos de treinos de musculação: treino para hipertrofia muscular, halterofilismo, potência muscular, resistência muscular e etc.

Atividade de Preferência

Rolamento dos ombros: O rolamento de ombros é outra opção, mais voltada para a parte superior do seu corpo. Levantando os seus ombros para cima e para baixo, você ativa a musculatura dos braços e tronco, se preparando para se exercitar.

Cuidados Pessoais

Comer bem: Se você praticar o esporte à noite, alimente-se bem durante o dia. Se você for praticar de manhã, turbine o seu jantar no dia seguinte. A alimentação saudável é elemento importante para melhorar o desempenho do atleta.
